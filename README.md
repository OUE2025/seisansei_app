# 生産性向上推進体制加算チェックアプリ

React + TypeScript + Vite で構成した、生産性向上推進体制加算の誤解解消・判定ウィザードと詳細チェックリスト（PDF出力付き）の単一ページアプリです。UI は Tailwind CSS と lucide-react を利用しています。

## 主な機能
- 誤解バスター → 簡易判定（S/A/B）→ プロモ/案内の 4 フェーズ構成
- 詳細チェックリスト（親子項目連動）と進捗バー、ローカル保存（チェックリストのみ保持）
- 判定S時の紙吹雪演出、S判定カード左右の飾り画像、結果メッセージの視認性向上
- PDF出力（jsPDF CDN＋ZenKakuGothicNew フォント読み込み）
- 公開用アセットは `public/`（各種イラスト・ロゴ類）

## 動作環境
- Node.js 18+ を推奨

## セットアップ & 実行
```bash
npm install          # 依存関係のインストール
npm run dev          # 開発サーバー（http://localhost:5173）
npm run build        # 型チェック付きビルド（dist 出力）
npm run preview      # ビルド済みをローカルプレビュー
```

## データ保持の仕様
- `localStorage` には「詳細チェックリスト」のチェック状態のみ保存されます。その他の回答やフェーズ進行はリロードでリセットされます。

## フォルダ構成（抜粋）
- `src/App.tsx`：アプリ本体。判定ロジック、フェーズ/チェックリストUI、PDF生成を実装。
- `src/index.css`：Tailwind レイヤーとカスタムアニメーション（紙吹雪、フェード等）。
- `public/`：トップロゴ、誤解カード画像、紙吹雪用飾りなどの静的アセット。

## 注意点
- PDF出力はネットワーク経由で jsPDF とフォントを取得します。オフライン環境ではエラーとなる場合があります。
- 画像パスは `import.meta.env.BASE_URL` を考慮したヘルパーで解決しています。サブパス配備でも動作する構成です。
